# Global Description
A small calculator app using SFML for rendering and input.\
Operations follow the standard PEMDAS order.\

# How To Compile
At the root of the repository:
```console
mkdir build
cd build
cmake ..
cmake --build .
# To launch
./CalculatorApp
```
